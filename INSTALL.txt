Aby zainstalować aplikację należy wykonać następujące kroki.
1. Zainstalować środowisko Oracle Java 7 lub nowsze.
2. Zainstalować serwer bazy danych MySQL.
3. Utworzyć bazę danych o nazwie semantyk.
4. W pliku resource/conf.properties uzupełnić konfigurację w celu poprawnego uruchomienia aplikacji.
5. Do katalogu database przekopiowac dumpy baz danych Wordnet oraz Wiktionary w postaci XML.
6. W celu otrzymania pomocy do aplikacji należy ją uruchumić bez żadnych parametrów.

To install application follow below steps:
1. Install JDK 7 or newer,
2. Install MySQL database server,
3. Create database with name semantyk,
4. In /resource/conf.properties fill up your configuration,
5. Copy plWordNet and Wiktionary dumps into database folder,
6. Run application without any arguments to get help.
